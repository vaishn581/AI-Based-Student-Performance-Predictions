# Frontend

This folder contains the user interface of the AI-Based Student Performance Prediction System.

## Files

- `index.html` – Main web page
- `style.css` – Stylesheet for designing the UI
- `script.js` – Handles user interaction and API requests

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Purpose

The frontend collects student details from the user and sends them to the Flask backend for prediction. The predicted result is then displayed on the webpage.
